# AI-Based-Crop-Disease-Detection
AI-Based Crop Disease Detection is a deep learning application that identifies plant diseases from leaf images using a CNN model. Users can upload a leaf image to receive the predicted disease, confidence score, and treatment suggestions. Built with Python, TensorFlow, Flask, HTML, CSS, JavaScript, and OpenCV to support smart agriculture.
